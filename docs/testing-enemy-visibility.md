## Тестовые сценарии

### Сценарий 1: Враг на открытой местности (должен атаковать)

**Подготовка:**
1. Запустить бота на равнине
2. Заспавнить зомби рядом (10-15 блоков)

**Ожидаемое поведение:**
- Лог: `👁️ [canSeeEnemy] zombie ВИДЕН (прямая видимость)`
- Лог: `⚔️ [analyzeCombat] Видимых врагов: 1, цель: zombie`
- Бот переходит в COMBAT
- Бот атакует зомби

**Проверка:**
- [ ] Бот обнаружил врага
- [ ] Бот перешёл в COMBAT
- [ ] Бот атакует

---

### Сценарий 2: Враг за стеной (не должен атаковать)

**Подготовка:**
1. Построить стену из камня (3x3x3)
2. Заспавнить зомби за стеной (8 блоков от бота)

**Ожидаемое поведение:**
- Лог: `🚫 [canSeeEnemy] zombie НЕ ВИДЕН (блокирует stone)`
- Лог: `🔍 [isEnemyReachable] Проверяю путь до zombie...`
- Лог: `❌ [isEnemyReachable] zombie НЕ ДОСТИЖИМ (нет пути)`
- Лог: `ℹ️ [updateEntities] Врагов обнаружено: 0 (после фильтрации)`
- Бот НЕ переходит в COMBAT

**Проверка:**
- [ ] Бот НЕ атакует врага за стеной
- [ ] Нет перехода в COMBAT
- [ ] Бот остаётся в IDLE/TASKS

---

### Сценарий 3: Враг за забором с путём обхода (должен атаковать)

**Подготовка:**
1. Построить забор (4 блока высотой)
2. Оставить проход в 15 блоках
3. Заспавнить зомби за забором (10 блоков от бота)

**Ожидаемое поведение:**
- Лог: `🚫 [canSeeEnemy] zombie НЕ ВИДЕН (блокирует oak_fence)`
- Лог: `🔍 [isEnemyReachable] Проверяю путь до zombie...`
- Лог: `✅ [isEnemyReachable] zombie ДОСТИЖИМ (путь: 25 блоков)`
- Лог: `⚔️ [updateEntities] Видимых/достижимых врагов: 1`
- Бот переходит в COMBAT
- Бот идёт к проходу и атакует

**Проверка:**
- [ ] Бот обнаружил путь
- [ ] Бот обошёл забор
- [ ] Бот атаковал врага

---

### Сценарий 4: Враг под землёй (не должен атаковать)

**Подготовка:**
1. Выкопать шахту вниз
2. Заспавнить зомби в соседней пещере (8 блоков по горизонтали)
3. Между ботом и зомби - земля/камень

**Ожидаемое поведение:**
- Лог: `🚫 [canSeeEnemy] zombie НЕ ВИДЕН (блокирует stone)`
- Лог: `🔍 [isEnemyReachable] Проверяю путь до zombie...`
- Лог: `❌ [isEnemyReachable] zombie НЕ ДОСТИЖИМ (нет пути)` 
  ИЛИ
- Лог: `⚠️ [isEnemyReachable] zombie НЕ ДОСТИЖИМ (путь 120 > макс 40)`
- Бот НЕ переходит в COMBAT

**Проверка:**
- [ ] Бот НЕ копает до врага
- [ ] Нет перехода в COMBAT
- [ ] Бот игнорирует врага

### Сценарий 5: Много врагов (производительность)

**Подготовка:**
1. Заспавнить 10 врагов в разных местах:
   - 3 на открытой местности
   - 4 за стенами без пути
   - 3 за стенами с путём обхода

**Ожидаемое поведение:**
- Raycast: 10 проверок (~10ms)
- Pathfinder: 7 проверок (только для невидимых) (~350-3500ms первый раз)
- Pathfinder с кешем: ~0.7ms второй раз
- Итого: 3 видимых + 3 достижимых = 6 врагов в context.enemies
- Бот атакует ближайшего из 6

**Проверка:**
- [ ] Не тормозит (FPS > 15)
- [ ] Бот атакует правильных врагов
- [ ] Кеш работает (второй раз быстрее)

---

### Сценарий 6: Кеш pathfinder

**Подготовка:**
1. Заспавнить зомби за стеной с путём обхода
2. Подождать 2 секунды (бот должен обнаружить)
3. Подождать ещё 2 секунды

**Ожидаемое поведение:**
- Первый раз: `🔍 [isEnemyReachable] Проверяю путь до zombie...`
- Второй раз (через 2 сек): `📦 [isEnemyReachable] zombie (из кеша: ДОСТИЖИМ)`
- Через 10 сек: `🧹 [cleanupPathfindCache] Удалено 1 устаревших записей`

**Проверка:**
- [ ] Кеш используется
- [ ] Автоочистка работает
- [ ] Производительность улучшена

## Ошибки компиляции

### Если TypeScript ругается на импорты:

```bash
# Проверить что алиасы настроены
npm run type-check

# Если ошибка "Cannot find module '@utils/combat/enemyVisibility'"
# Проверить tsconfig.json paths
```

### Если ошибка "Property 'pathfindTimeout' does not exist"

Проверить что:
1. Обновлен `src/hsm/context.ts`
2. Добавлены поля в интерфейс `MachineContext['preferences']`
3. Добавлены значения по умолчанию в `context.preferences`

## Troubleshooting

### Бот всё равно атакует врагов за стенами
- Проверить что используется обновлённый `updateEntities` action
- Проверить логи - вызывается ли `canSeeEnemy`?
- Проверить guard `isEnemyNearby` - использует ли `context.enemies`?

### Бот не атакует никого
- Проверить `context.preferences.maxDistToEnemy` (может быть слишком маленький)
- Проверить логи - есть ли враги после фильтрации?
- Проверить guard `isEnemyNearby` - правильно ли проверяет `context.enemies.length`?

### Ошибка "bot.world.raycast is not a function"
- Проверить версию mineflayer (должна быть >= 4.0)
- Обновить: `npm install mineflayer@latest`

### Pathfinder не находит пути
- Проверить что pathfinder инициализирован
- Проверить что `bot.movements` настроен
- Увеличить `pathfindTimeout` до 2000ms
