# Вклад в проект

Языковые версии: [English](CONTRIBUTING.md) | [Русский](CONTRIBUTING.ru.md)

Этот репозиторий публичный, но это не место для случайных правок. Изменения должны быть небольшими, проверяемыми и согласованными с существующей архитектурой.

## Перед открытием PR

1. Прочитайте релевантную документацию в `docs/`.
2. Запустите `npm run type-check`.
3. Запустите `npm run build`.
4. Запустите focused tests для затронутого подсистемного кода.

## Ожидания

- Следуйте существующим соглашениям TypeScript, XState и Mineflayer.
- Добавляйте или обновляйте тесты, если меняется поведение.
- Не вводите обходные пути для persistence, скрытое глобальное состояние или логику, которая обходит HSM.
- Держите изменения в рамках того подсистемного блока, к которому они относятся.

## Структура репозитория

- Runtime code: `src/`
- Tests: `src/tests/`
- Documentation: `docs/`
- Runtime memory: `data/`
- Logs: `logs/`

## Pull Request

Используйте шаблон PR в `.github/PULL_REQUEST_TEMPLATE.md` и укажите:

- краткое описание
- изменение поведения
- шаги проверки
- любые риски или follow-up
